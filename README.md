# Wishlist

## 개요
- 오프라인 쇼핑을 위한 장바구니
- 안정적인 테스트를 위해, Desktop 앱으로 먼저 개발
- UI는 챗GPT의 도움으로 작성

## UI 설명
- 쇼핑 정보 리스트
  - 리스트 항목을 클릭하면 수정 페이지로 이동
  - 하단의 + 버튼을 클릭하면 입력 페이지로 이동
  <img width="1266" height="713" alt="image" src="https://github.com/user-attachments/assets/5d376eaa-4cb5-4161-9728-a8d1840829ef" />

- 쇼핑 정보 입력
    - 쇼핑 장소는 직접 입력
    - 쇼핑 날짜는 <날짜 선택>을 클릭하여 Datepicker로 입력: YYYY-MM-DD 형식
    - 수정 페이지도 레이아웃은 동일
  <img width="1266" height="713" alt="image" src="https://github.com/user-attachments/assets/6ef1819a-8b84-495d-9543-abcff5277c95" />


- 쇼핑 상세보기
  - 구매할 품목과 구매여부, 쇼핑에 대한 상세 정보 확인
  - 오른쪽 체크박스를 클릭하여 구매 여부 체크
  <img width="1266" height="713" alt="image" src="https://github.com/user-attachments/assets/34a39191-a1d3-4085-b5fc-6175c5a71c10" />

- 품목 추가
  - 품목명, 구매량, 단가로 구성
  - 모두 자유 입력
